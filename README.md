# Copy Paste Shadows

A Plugin for copying and pasting shadows across layers.

## Install

Download or clone the repository, and double click copy-paste-shadows.sketchplugin

## Development Guide

* Initial setup

    ```bash
    $ npm run setup
    ```

* Build plugin

    ```bash
    $ npm run build
    ```

* To watch for changes

    ```bash
    $ npm run watch
    ```
